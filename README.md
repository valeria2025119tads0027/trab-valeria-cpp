# trab-valeria-cpp

Esse sistema para PetShop foi desenvolvido na linguagem em C++, utilizando os pilares da Programação Orientada a Objetos. Ele funciona como um console administrativo onde é controlado desde a entrada do pet até o fechamento do caixa. É possível cadastrar informações do Pet, tem um catálogo de serviços, histórico de serviços prestados e há o controle financeiro. Foi feito o possível para abordar todos os conteúdos passados durante esse período da disciplina de Programação Orientada a Objetos.
